# iam-access-management-lab
This project simulates an Identity and Access Management (IAM) environment for a fictional company

The goal is to demonstrate IAM concepts such as:

• Role-Based Access Control (RBAC)
• Least Privilege Access
• User Onboarding and Offboarding
• Access Reviews
• Identity Lifecycle Management

Tools/Concepts Used:
- GitHub for identity simulation
- RBAC models
- Access policy documentation

This lab demonstrates practical IAM workflows similar to those used in enterprise environments.
